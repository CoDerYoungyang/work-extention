# 牛马小助手

一个简单的Chrome浏览器插件，帮助你计算工作时间和工资进度。

## 功能特点

- 上下班时间设置
- 实时下班倒计时
- 当日工资进度计算
- 数据本地保存

## 使用方法

1. 打开Chrome浏览器，进入扩展程序页面（chrome://extensions/）
2. 开启开发者模式
3. 点击"加载已解压的扩展程序"
4. 选择本项目文件夹
5. 点击插件图标，设置你的工作时间和薪资信息

## 文件结构

- manifest.json: 插件配置文件
- popup.html: 插件弹出窗口
- popup.js: 主要功能逻辑
- styles.css: 样式文件
